I developed a Python program using only numpy and PIL libraries to apply the following transformations to an image (e.g., squares.png, bird.png):

1.1) Squares.png
a. Similarity transformation: (tx=10, ty=20,,s = 2, theta=30 degree )
b. Affine transformation (shx=20, shy=10)
c. Change parameters to transform the input image to the output image to see
the effect of parameters.
c1) Similarity transformation: (tx=20, ty=10,,s = 3, theta=45 degree )
c2) Affine transformation (shx=10, shy=20)

1.2) Bird.png
a. Similarity transformation: (tx=10, ty=20,,s = 2, theta=30 degree )

the effect of parameters.
